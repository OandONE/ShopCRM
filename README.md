# ShopCRM — نسخه Alpha (پیش‌نمایش اولیه)

یک CRM ساده، سبک و متن‌باز برای **مغازه‌های کوچک و فروشگاه‌های محلی**.

> ⚠️ **این نسخه Alpha است — ناپایدار، ناقص، و پر از باگ.**

---

## ⚠️ هشدار مهم

این پروژه در مرحله **Alpha** قرار دارد. بسیاری از بخش‌ها ناقص هستند و باگ‌های زیادی در آن وجود دارد — از باگ‌های ریز UI گرفته تا مشکلات جدی در backend. برخی از باگ‌ها را خودم می‌دانم و هنوز برطرف نشده‌اند.

**از این نسخه برای استفاده واقعی و تجاری استفاده نکنید.**

---

## 🐛 وضعیت فعلی

- ✅ مفاهیم اولیه کار می‌کنند
- ✅ می‌توان فروش ثبت کرد
- ✅ جستجوی محصول کار می‌کند
- ❌ بسیاری از بخش‌ها ناقص هستند
- ❌ باگ‌های متعدد در UI و backend
- ❌ امنیت کامل نیست
- ❌ مستندات کامل نیست

---

## 🚀 نصب و اجرا

```bash
git clone https://github.com/OandONE/ShopCRM.git
cd ShopCRM
pip install -r requirements.txt
python main.py
```

سپس `http://127.0.0.1:8000` را باز کنید.

**ورود به پنل ادمین:** `admin` / `admin`

---

## 🛠️ تکنولوژی‌ها

- **Backend:** FastAPI + SQLModel
- **Database:** SQLite
- **Frontend:** Jinja2 + Vanilla JS
- **UI:** Dark Mode پشتیبانی می‌شود

---

## 📋 ویژگی‌های فعلی

- فرم فروش سریع و کامل
